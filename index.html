Multiple Object detection using pre-trained model

Difficulty: Easy

How to use

Wait for the pre-trained model to load before use

Click on an image below to view an object with even an importance that you can see the position and how many the objects are.

Hold some objects close to your webcam to get a classification. You must be on the https version of the website for this work. Don't forget to enable your webcam.
